# Avalon Parfums — Landing Page

Landing page da Avalon Parfums (perfumaria de nicho, importados e árabes — Baixada Santista),
construída com Next.js (App Router) + Tailwind CSS.

> Este repositório foi reconstruído a partir do conteúdo publicado em
> [avalon-parfums-qr.vercel.app](https://avalon-parfums-qr.vercel.app) para conectar o projeto
> ao GitHub e habilitar deploys via Git na Vercel.

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

## Deploy

Projeto conectado à Vercel: **avalon-parfums-qr**.
